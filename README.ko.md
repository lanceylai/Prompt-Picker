Prompt Picker는 제품 관리자와 개발자가 페이지에서 수정해야 할 내용을 명확하고 실행 가능한 AI 코딩 프롬프트로 바꿀 수 있게 도와주는 Chrome 확장 프로그램입니다.

"카드 오른쪽에 있는 버튼"처럼 애매하게 설명하지 않아도 됩니다. 페이지에서 정확한 요소를 클릭하고, 변경 요청을 작성한 뒤, Claude Code, Codex, Cursor 또는 다른 AI 코딩 도구에 붙여넣을 수 있는 구조화된 프롬프트를 복사할 수 있습니다.

## ✨ 무엇을 할 수 있나요?

Prompt Picker는 다음을 도와줍니다.

1. 수정이 필요한 요소 선택: 페이지에서 대상 요소를 직접 선택하고 수정 요구사항을 설명합니다.
2. 컨텍스트 Prompt 자동 생성: Selector, DOM 구조, 컴포넌트 정보, 페이지 컨텍스트가 포함된 AI Prompt를 생성합니다.
3. AI 수정 성공률 향상: AI Agent가 해당 코드를 빠르게 찾도록 돕고, 반복 커뮤니케이션을 줄이며 수정 정확도를 높입니다.

## 👥 누구에게 유용한가요?

| 역할 | Prompt Picker가 돕는 방식 |
| --- | --- |
| 제품 관리자 | UI 변경 요청, 상호작용 문제, 제품 개선 의견을 페이지의 정확한 위치와 함께 더 명확하게 전달할 수 있습니다. |
| 개발자 | AI 코딩 도구에 정확한 컨텍스트를 제공해 어떤 요소를 수정해야 하는지, 관련 코드를 어디서 찾아야 하는지 전달할 수 있습니다. |

## 💡 자주 쓰는 상황

- 정확한 UI 맥락이 포함된 제품 피드백 issue를 만들 때.
- AI Agent에게 버튼, 카드, 메뉴, 폼, 섹션 수정을 요청할 때.
- 제품 리뷰 중 변경 요청을 빠르게 기록할 때.
- "이 부분이 이상하다"는 느낌을 실행 가능한 작업으로 바꿀 때.
- 여러 페이지에 걸친 플로우의 수정 지점을 모을 때.

## 🧭 사용 방법

1. Chrome Web Store에서 Prompt Picker를 설치합니다.
2. 검토하려는 웹 페이지를 엽니다.
3. 브라우저 툴바의 Prompt Picker 아이콘을 클릭하거나 단축키를 사용합니다. Mac은 `Option + Q`, Windows는 `Alt + Q`입니다.
4. 요소를 클릭하거나 드래그해서 여러 요소를 선택합니다.
5. 변경 요청을 작성합니다.
6. **Pick & Copy**를 클릭합니다.
7. 복사된 프롬프트를 AI 코딩 도구에 붙여넣습니다.

## 🎯 왜 도움이 되나요?

AI 코딩 도구는 정확한 맥락을 받을수록 더 잘 동작합니다. Prompt Picker는 다음 정보를 프롬프트에 담습니다.

- 선택한 페이지 요소.
- 유용한 DOM 및 selector 정보.
- 주변 텍스트와 구조.
- 사용자의 변경 요청.

이를 통해 제품과 개발 사이의 반복 커뮤니케이션을 줄이고, AI Agent가 더 정확하게 코드를 수정할 수 있습니다.

## 🔒 개인정보

Prompt Picker는 브라우저 안에서 동작합니다. 선택한 페이지 맥락을 수집해 클립보드에 복사하며, 어디에 붙여넣을지는 사용자가 직접 결정합니다.
