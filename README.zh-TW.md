Prompt Picker 是一個 Chrome 擴充功能，幫助產品經理和研發工程師把頁面上需要修改的內容，變成清楚、可執行的 AI 程式提示詞。

你不需要再說「右邊卡片裡的那個按鈕」這種模糊描述。用 Prompt Picker 直接點選頁面元素，寫下修改說明，然後複製一段結構化 prompt，貼給 Claude Code、Codex、Cursor 或其他 AI 程式工具。

## ✨ 它能做什麼

Prompt Picker 可以幫助你：

1. 選擇需要修改的元素：在頁面中直接選中目標元素，並描述你的修改需求。
2. 自動產生上下文 Prompt：產生包含 Selector、DOM 結構、元件資訊和頁面上下文的 AI Prompt。
3. 提升 AI 修改成功率：幫助 AI Agent 快速找到對應程式碼，減少反覆溝通，提高修改準確率。

## 👥 適合誰使用

| 角色 | Prompt Picker 如何幫助你 |
| --- | --- |
| 產品經理 | 描述 UI 修改、互動問題或產品建議時，直接點選頁面中的具體位置，讓回饋更清楚。 |
| 研發工程師 | 為 AI 程式工具提供準確上下文，讓它理解「到底要改哪個元素」以及去哪裡找相關程式碼。 |

## 💡 常見場景

- 建立帶有明確 UI 上下文的產品回饋 issue。
- 讓 AI Agent 修改某個按鈕、卡片、選單、表單或頁面區塊。
- 在產品評審時快速記錄修改意見。
- 把「這裡感覺不對」變成更可執行的任務描述。
- 跨多個頁面收集同一個流程裡的修改點。

## 🧭 使用方法

1. 從 Chrome Web Store 安裝 Prompt Picker。
2. 開啟你要檢視的網頁。
3. 點擊瀏覽器工具列裡的 Prompt Picker 圖示，或使用快速鍵：Mac 使用 `Option + Q`，Windows 使用 `Alt + Q`。
4. 點擊元素，或拖曳框選多個元素。
5. 寫下你的修改說明。
6. 點擊 **Pick & Copy**。
7. 把複製出的 prompt 貼到 AI 程式工具裡。

## 🎯 為什麼有用

AI 程式工具拿到的上下文越準確，越容易改對地方。Prompt Picker 會把這些資訊整理進 prompt：

- 你選中的頁面元素。
- 有用的 DOM 和 selector 資訊。
- 附近文字和結構。
- 你的修改說明。

這樣可以減少產品和研發之間的反覆溝通，也能讓 AI Agent 更有針對性地修改程式碼。

## 🔒 隱私說明

Prompt Picker 在瀏覽器內執行。它會收集你選中的頁面上下文，並複製到剪貼簿，由你決定貼到哪裡。
